## Random Quote Generator

A clean, responsive web application that provides users with a "Quote of the Day" at the click of a button. This project uses a third-party API to fetch fresh advice and inspiration, featuring a modern UI built with HTML, CSS, and Vanilla JavaScript.

---

### ### Features

* **Dynamic Quote Fetching**: Retrieves real-time advice using the Advice Slip JSON API.
* **Copy to Clipboard**: One-click functionality to copy the current quote for easy sharing.
* **Responsive Design**: A centered, card-based layout that works across various screen sizes.
* **Loading States**: Visual feedback on the button while a new quote is being retrieved.
* **Error Handling**: Graceful fallback message if the API service is temporarily unavailable.

---

### ### File Structure

* **`index.html`**: Defines the structure, including the quote display area and action buttons.
* **`style.css`**: Handles the aesthetic, utilizing the Poppins font and a vibrant blue theme.
* **`script.js`**: Manages the logic for API calls and the clipboard functionality.

---

### ### How to Use

1.  **Open the App**: Launch `index.html` in any modern web browser.
2.  **Get Inspired**: Click the **New Quote** button to fetch a fresh piece of advice.
3.  **Share**: Click the **Copy icon** (circular button on the left) to copy the text to your clipboard.

---

### ### Technical Implementation

* **Icons**: Font Awesome (v5.15.3).
* **Fonts**: Google Fonts (Poppins).
* **Styling**: Flexbox for centering and alignment.
